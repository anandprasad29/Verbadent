# Font Setup Instructions

To use the Kumar One font in this Flutter project:

1. **Download the font:**
   - Visit [Google Fonts - Kumar One](https://fonts.google.com/specimen/Kumar+One)
   - Click "Download family"
   - Extract the ZIP file

2. **Add the font file:**
   - Copy `KumarOne-Regular.ttf` from the extracted folder
   - Paste it into the `fonts/` directory in this project

3. **Verify the setup:**
   - The font should be at: `fonts/KumarOne-Regular.ttf`
   - The `pubspec.yaml` is already configured to use this font

4. **Run the app:**
   ```bash
   flutter pub get
   flutter run
   ```

The font is configured in `pubspec.yaml` and will be automatically available throughout the app via the theme.
